Teleport!

March 2018, CS 61B

Over the course of 3 weeks, my partner, Leo Cai, and I created a 2D random world generation game with little guidance. This game uses keyboard controls to move the player around a random world. The teleporter leads to a new randomly generated world. This game incorporates music and colorful visuals. The game can be played in English or Chinese. This project can be found on Github. Project specs can be found on this site.







This project showed us how to create a beautiful product from scratch. We had nothing to start off with, but through data abstraction and object oriented programming, we built each piece of the game. We spent hours debugging, and in the end, it was a very rewarding experience.

Add One

June 2017, LearnAppMaking

I attended an Apple iOS workshop over the summer, where I learned basic Swift. I used the graphics the website provided and made this app using my own Swift knowledge and LearnAppMaking resources. In this game, you add one to every number. You have one minute to do as many as possible. My code can be found on Github.



I learned how to apply Swift to a game with a timer, a score keeper, and interesting graphics. It was interesting to figure out how to make sure the app looks good on all devices and in different orientations. This gave me the confidence to make my own apps from scratch, since I was able to go through the entire iOS app developing process.

Ball of Fortune

August 2014, Team Treehouse

During my freshman year of high school, the app Flappy Bird was extremely popular. I was inspired by the success of such a simple app to create my own apps. Over the summer, I went through an online Android course by Team Treehouse. Through this course, I was able to create this fortune telling app.



I learned how to develop an Android app from start to finish. I became familiar with using Eclipse and began to understand how to write the code for linking an action, like shaking the phone, to a certain part of the code. I also figured out how to publish apps to the Google Play Store.

DomainateTheWorld

April 2017, AthenaHacks

Due to other time-conflicting commitments, I was only able to attend USC's AthenaHacks for a few hours.
