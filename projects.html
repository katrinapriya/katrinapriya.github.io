Innovation Database

Fall 2019, CS 169

In our software engineering class, we developed SaaS for an external customer, while learning Agile practices. Over the course of 3 weeks, my partner and I created a 2D random world generation game with little guidance. This game uses keyboard controls to move the player around a random world. CPU Design

December 2018, CS 61C

Using logisim, my partner and I implemented a 32-bit two-cycle processor for RISC-V for our computer architecture class. Our processor implemented over 30 RISC-V instructions using control signals and 2-stage pipelining. We used Logisim gates, multiplexers, shifters, priority encoders, bit extenders, splitters and more.



This was a super cool project because we got to learn basically how a computer is built, down to the circuit elements.
